CardioResp Escape

Este jogo foi feito em Python + Pygame e roda no computador.

Como rodar

  1. Baixe o projeto (ZIP) do GitHub.
  2. Abra a pasta no Windows.
  3. Crie ambiente virtual e instale dependências.
  4. Rode: python main.py
